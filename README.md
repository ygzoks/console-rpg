# Console RPG

A turn-based RPG game running in the terminal, written in C++.
Built as a learning project to practice object-oriented programming concepts.

## Gameplay

Fight against bosses in turn-based combat, earn gold, and grow stronger between rounds.

- **Attack** — Deal damage to the boss
- **Heavy Attack** — High-risk, high-reward strike (costs more stamina)
- **Defend** — Reduce incoming damage by half
- **Potion** — Restore HP (limited uses)

## Features

- Turn-based combat system with stamina management
- Boss AI with multiple behaviors (attack, heavy attack, defend)
- Critical hit system for both player and boss
- Level scaling — bosses get stronger each round
- Shop system — spend gold to upgrade damage or restore HP
- Color-coded terminal UI with dynamic HP display

## Tech

- Language: C++
- IDE: Visual Studio
- No external libraries — standard library only

## Status

Active development. UI improvements and new mechanics planned.
